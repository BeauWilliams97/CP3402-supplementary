PressBook Dark is a multi-purpose dark WordPress theme suitable for blog, business, marketing, news, magazine, and educational website.

== Description ==

PressBook Dark theme offers multiple eye-catching block patterns, single and double sticky sidebars that float on scrolling, footer widgets, header and footer blocks to insert any Gutenberg block for custom content, top banner area, multi-level dropdown menu, social links menu, page builder compatibility, an easy-to-read dark color scheme, highly engaging user interface design based on advanced typography techniques, and vibrant colors combined with the best contrast ratio. This theme is a great choice for publishing tutorials, posting guides and documentation, posting reviews and opinions, movies, anime, or manga reviews, technology, devices, and gadgets reviews, events feed, story feeds, status updates, newsfeed, etc. This theme can also be used to create gaming sites, music streaming, corporate, consulting, or entertainment websites.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

PressBook Dark includes support for WooCommerce and for Infinite Scroll in Jetpack.

= Where to find the theme options? =

All of the options can be found in the Customizer in 'Appearance > Customize'.

= How to configure the theme widgets? =

The theme widgets can be found in 'Appearance > Widgets'. You can drag and drop widgets to the widget area. PressBook Dark comes with 6 widgets area locations which include a left sidebar, right sidebar widgets area and 4 locations for footer widgets.

= How to add the menus or social links? =

The menus can be found in 'Appearance > Menus'. First, create the menu and then assign it to a menu location. PressBook Dark comes with three menu locations which include the primary menu, top bar menu, and social links menu. For social links, create a menu item with a custom link, enter the URL of the social page, and assign this menu to the 'Social Links Menu' location.
